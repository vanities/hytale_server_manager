## Overview

The Mod Marketplace integrates with Modtale to provide access to mods for your Hytale servers. Install, update, and manage mods without leaving the panel.

![Server Management](/hytale_server_manager/images/mod_marketplace.jpeg)

## Browsing Mods

### Accessing the Marketplace

1. Navigate to **Marketplace** in the sidebar
2. Browse the available mods
3. Use filters and search to find specific content

### Filter Options

- **Type**: Plugins, Data Assets, Art Assets, World Saves
- **Sort By**: Downloads, Rating, Title, Author
- **Search**: Search by name or description

### Mod Information

Each mod displays:

- Name and author
- Download count
- User rating
- Description
- Version information
- Tags

## Installing Mods

### One-Click Installation

1. Find the mod you want to install
2. Click the **Install** button
3. Select the target server
4. Wait for installation to complete

### Installation Queue

When installing multiple mods:

- View the installation queue
- Track progress for each mod
- Cancel pending installations

## Managing Installed Mods

### Viewing Installed Mods

1. Go to your server's detail page
2. Click on **Installed Mods**
3. View all mods installed on that server

### Uninstalling Mods

1. Find the mod in your installed list
2. Click the **Uninstall** button
3. Confirm the removal

## API Configuration

### Setting Up Modtale API

To use the marketplace, configure your Modtale API key:

1. Go to **Settings**
2. Enter your Modtale API key
3. Save the configuration

### Getting an API Key

Visit [Modtale](https://modtale.net) to create an account and generate an API key.

## Best Practices

1. **Test First**: Try mods on a test server before production
2. **Check Compatibility**: Verify mod compatibility with your server version
3. **Backup Before Installing**: Always backup before adding new mods
4. **Read Reviews**: Check ratings and reviews before installing
